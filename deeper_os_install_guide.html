DEEPER OS · 설치 ②

디퍼 OS 깔기

클로드 코드를 켰다면, 이제 내 볼트에 디퍼 OS(사막여우)를 깔 차례예요. 명령 몇 줄이면 돼요 — 한 줄씩 천천히 따라오세요. 🦊

시작 전 두 가지 확인

✅ 옵시디언 볼트를 만들어 뒀고  ✅ 클로드 코드 Code 탭을 켜뒀나요? 아직이면 이것부터 —
볼트 만들기 가이드  ·  클로드 코드 설치 가이드(①)

PART 1

내 볼트 연결하기

사막여우가 내 노트를 보려면, “이 폴더에서 일해”라고 가리켜줘야 해요.

1
볼트 폴더 열기

Code 탭에서 Select folder → 옵시디언으로 만든 그 볼트 폴더를 골라요.

2
잘 연결됐는지 확인

입력창에 여기 어떤 폴더야? 안에 뭐가 있어? 라고 쳐서, 내 노트를 읽어주면 — 연결 OK!

PART 2

터미널에서 디퍼 OS 깔기

천천히, 한 줄씩. 칠 건 몇 줄뿐이에요. 틀려도 다시 치면 돼요.

3
터미널 열기

Code 탭 화면 오른쪽 위 >_ 버튼, 또는 Ctrl + 백틱(맨 왼쪽 위, 물결 ~과 같은 키). → 아래(또는 오른쪽)에 터미널 칸이 열려요.

⚠️ 이건 별도 프로그램이 아니라 클로드 앱 안의 터미널이에요(맥·윈도우 똑같아요).

4
터미널에 claude 한 단어만

터미널 칸에 claude → Enter. (아직 슬래시 명령 아니에요.)

“claude 인식 안 됨 / not recognized”가 뜨면 한 번만 설치:

npm install -g @anthropic-ai/claude-code

→ 끝나면 다시 claude → Enter.

5
첫 실행 화면 넘기기 (거의 Enter)

테마(색) → Enter · 로그인(누르면 브라우저가 열려요 → 유료 계정 로그인 → ‘허용/Authorize’ → 터미널로 돌아오기) · 깃허브 연결 화면 → Skip · 보안 안내 → Enter · “이 폴더를 신뢰?” → Yes.

→ 화면이 바뀌고 아래에 > 입력 표시가 뜨면, 이제 여기가 ‘클로드 안’이에요. 🦊

📍 제일 헷갈리는 곳! 어디에 치는지 — 세 칸이 달라요
① ❌Code 탭 채팅창 — 여기선 /plugin이 안 돼요(“isn’t available”).
②터미널 칸(방금 연 것) — 여기엔 claude만 쳤어요.
③ ✅그 안에서 실행된 ‘클로드’(> 표시) — 아래 명령은 꼭 ③ 여기에.
👉 터미널 열기 → claude 실행 → 실행된 클로드 안(>)에 /plugin…
6
클로드 안(>)에서 세 줄, 한 줄씩 Enter
/plugin marketplace add https://github.com/clbookslower/deeper-os.git
/plugin install deeper-os@deeper-os
/reload-plugins

중간에 “추가/설치할까요?” 물으면 Yes / Enter.

7
확인 — 사막여우 깨우기

같은 자리(클로드 안)에서:

/deeper-os:setup

사막여우가 “제가 뭐라고 불러드릴까요?” 하고 인사하며 시작하면 — 설치 성공! 🎉

⚠️ /plugin이 “isn’t available”라고 뜨면

채팅창에 친 거예요. Code 탭 채팅창이 아니라, 터미널의 claude 안(> 표시 보이는 곳)에서 다시 쳐주세요. (위 ‘세 칸’ 참고.)

🔄 이미 디퍼 OS를 쓰던 분

다시 설치 말고, 클로드 안에서 /plugin marketplace update deeper-os → /reload-plugins 로 업데이트만. (안 바뀌면 + → Plugins에서 삭제 후 재설치. 노트는 그대로예요.)

디퍼 OS 깔기 끝! 🦊

/deeper-os:setup 으로 사막여우가 보금자리를 다 지으면, 이제 매일 함께예요.
