DEEPER OS · 설치 ②

디퍼 OS 깔기

클로드 코드를 내 볼트에 연결했다면, 이제 사막여우를 깔 차례예요. 명령은 딱 세 줄 — 한 줄씩, 아주 천천히 같이 해요. 틀려도 다시 치면 돼요. 🦊

시작 전 확인

✅ 옵시디언 볼트를 만들었고  ✅ 클로드 코드가 그 볼트에 연결돼 있나요? (앞 단계 = 클로드 코드 설치 ①) 됐으면 시작해요.

🖥️ ‘터미널’이 처음이세요? (30초만)

터미널은 글로 명령을 시키는 칸이에요. 무섭지 않아요 — 한 줄 치고 Enter가 전부예요. 비밀번호처럼 글자가 안 보여도 정상이고, 틀려도 다시 치면 돼요. 붙여넣기는 🪟 마우스 우클릭(또는 Ctrl+V) · 🍎 Cmd+V.

1
터미널 열기 (앱 안에서)

두 가지 방법 중 아무거나 —

  • 버튼: Code 탭 화면 오른쪽 위 >_ 아이콘(터미널 모양)을 클릭.
  • 단축키: Ctrl + 백틱(맨 왼쪽 위, 숫자 1 왼쪽 · 물결 ~과 같은 키). 🍎 맥도 Ctrl+백틱(Cmd 아니에요).

→ 화면 아래(또는 오른쪽)에 터미널 칸이 열려요. 위에 “터미널”이라고 적혀 있어요.

⚠️ 이건 윈도우 PowerShell·맥 ‘터미널’ 같은 별도 프로그램이 아니라, 클로드 앱 안에 있는 터미널이에요.

2
터미널에 claude 한 단어만

방금 연 터미널 칸에 claude 라고 치고 Enter. (아직 슬래시 명령 아니에요, 딱 이 한 단어.)

대부분 바로 설정 화면이 떠요 → 3번으로.

“claude 인식 안 됨 / not recognized”가 뜨면

딱 한 번만 이걸 치고 Enter (설치돼요, 몇 분):

npm install -g @anthropic-ai/claude-code

→ 끝나면 다시 claude → Enter.

3
첫 실행 화면 넘기기 (거의 Enter)

처음 한 번만 설정이 떠요. 거의 Enter로 통과해요 —

  • 테마(색) → 고르고 Enter.
  • 로그인 → 누르면 브라우저가 열려요 → 유료 계정 로그인 → ‘허용/Authorize’ → 터미널로 돌아오기.
  • 깃허브 연결 화면이 나오면 → 지금은 Skip/나중에.
  • 보안 안내 → Enter.
  • “이 폴더를 신뢰?” → Yes / Enter.

→ 화면이 바뀌고 아래에 > 입력 표시가 뜨면, 이제 여기가 ‘클로드 안’이에요(터미널이 클로드로 바뀐 거예요). 🦊

📍 제일 헷갈리는 곳! 어디에 치는지 — 세 칸이 달라요
① ❌Code 탭 채팅창 — 여기선 /plugin이 안 돼요(“isn’t available”).
②터미널 칸(1번에서 연 것) — 여기엔 claude만 쳤어요.
③ ✅그 안에서 실행된 ‘클로드’(> 표시 보이는 곳) — 아래 명령은 꼭 ③ 여기에.
👉 터미널 열기 → claude 실행 → 실행된 클로드 안(>)에 /plugin…
💡 폰으로 보는 중이면 — 주소가 짤려요

설치하는 그 PC에서 이 가이드를 열면 훨씬 쉬워요. 아래 명령을 마우스로 드래그 → 복사(Ctrl+C) 해서 클로드에 붙여넣기(Ctrl+V) 하면 주소가 짤릴 일이 없어요. 폰으로 보고 PC에 옮겨 치면 긴 주소가 잘려 자꾸 실패해요.

4
클로드 안(>)에서 세 줄, 한 줄씩 Enter

한 줄 치고 Enter, 결과 뜨면 다음 줄 — 이렇게 천천히. 첫 줄 주소는 길어요 — 맨 끝 .git 까지 전부 들어가야 해요.

/plugin marketplace add https://github.com/clbookslower/deeper-os.git

↳ 디퍼 OS가 있는 ‘가게(마켓플레이스)’를 등록해요.

/plugin install deeper-os@deeper-os

↳ 그 가게에서 디퍼 OS를 받아 깔아요.

/reload-plugins

↳ 방금 깐 걸 켜요(새로고침).

중간에 “추가/설치할까요?” 물으면 Yes / Enter.

5
확인 — 사막여우 깨우기

같은 자리(클로드 안)에서 이걸 치고 Enter:

/deeper-os:setup

사막여우가 “제가 뭐라고 불러드릴까요?” 하고 다정하게 인사하며 시작하면 — 설치 성공! 🎉 (사막여우가 폴더·노트 틀을 ‘만들까요?’ 물으면 네 해주세요. 기존 노트는 안 건드려요.)

⚠️ /plugin이 “isn’t available”라고 뜨면

채팅창에 친 거예요. Code 탭 채팅창 말고, 터미널의 claude 안(> 표시 보이는 곳)에서 다시 쳐주세요. (위 ‘세 칸’ 참고.) — 이게 제일 흔한 실수라, 이것만 맞으면 거의 다 돼요.

🔄 이미 디퍼 OS를 쓰던 분

다시 설치 말고, 클로드 안에서 /plugin marketplace update deeper-os → /reload-plugins 로 업데이트만. (안 바뀌면 + → Plugins에서 삭제 후 재설치. 노트는 그대로예요.)

디퍼 OS 깔기 끝! 🦊

사막여우가 보금자리를 다 지으면, 이제 매일 함께예요.
