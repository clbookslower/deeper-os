DEEPER OS · 백업 준비

깃허브(GitHub) 준비하기

디퍼 OS는 내 노트를 ‘깃허브’라는 안전한 금고에 백업해요(비공개라 나만 봐요). 그러려면 ① 깃허브 계정과 ② PC에 Git이 한 번 필요해요. 무료고, 천천히 따라오면 돼요. 🦊

PART 1

깃허브 계정 만들기 (무료·5분)

웹사이트에서 가입만 하면 돼요. 폰·PC 어디서든 되지만 PC로 하시길 권해요. 화면이 영어로 떠도 당황 마세요 — 아래 영어 버튼 이름 그대로 누르면 돼요.

1
깃허브 사이트 접속 → 가입

github.com 에 들어가 오른쪽 위 Sign up(가입) 을 눌러요.

2
이메일 입력

인증 코드를 받을 수 있는 이메일을 적고 Continue.

3
비밀번호 만들기

비밀번호를 정하고 Continue. 꼭 어딘가에 적어두세요 — 나중에 백업 연결할 때 써요.

4
사용자 이름(Username) 정하기

영문·숫자로 된 이름을 정해요. 이미 쓰는 사람이 있으면 다른 걸로. (공개되는 이름이에요. 닉네임처럼.) → Continue.

5
사람인지 확인 → 계정 만들기

간단한 퍼즐을 풀고, Create account를 눌러요.

6
이메일 인증 코드 입력

적은 이메일로 숫자 코드가 와요. 그 코드를 입력하면 — 가입 완료! 🎉 (가입 후 설문이 나오면 대충 고르거나 건너뛰어도 돼요.)

⚠️ 요금제(플랜)를 고르라고 하면

꼭 Free(무료)를 고르세요. 디퍼 OS 백업은 무료 비공개 저장소면 충분해요 — 돈 낼 필요 없어요. (가입 직후 나오는 설문이나 플랜 화면은 Free 선택 또는 건너뛰기.)

적어둘 두 가지

사용자 이름(Username)과 비밀번호 — 이 둘은 디퍼 OS 백업 연결할 때 쓰니 메모해 두세요.

PART 2

PC에 Git 깔기 (백업 도구)

디퍼 OS가 백업할 때 ‘Git’이라는 도구를 써요. 보통은 디퍼 OS 세팅이 자동으로 깔아줘요 — 자동이 안 될 때만 아래처럼 직접.

먼저 — 보통은 자동

디퍼 OS 세팅에서 백업 연결 이라고 하면, 사막여우가 Git을 알아서 깔아줘요. 그럼 이 PART 2는 건너뛰어도 돼요. 자동이 안 되거나 “git을 못 찾아요”가 뜨면 ↓

1
🪟 윈도우 Git for Windows 받기

git-scm.com 접속 → Download for Windows → 받은 파일을 실행.

2
🪟 윈도우 그냥 ‘Next’ 연타 → Install

설치 화면이 여러 개 떠도 전부 기본값 그대로 Next 만 누르면 돼요. 마지막에 Install → Finish. (어려운 선택 안 해도 돼요.)

·
🍎 맥 보통 이미 있어요

맥은 Git이 대개 깔려 있어요. 없으면 디퍼 OS 백업 연결 때 “명령어 도구 설치” 창이 뜨는데, 설치를 누르면 돼요(몇 분).

계정 + Git 준비 끝! ✅

이제 디퍼 OS에서 백업 연결 이라고 하면, 방금 만든 계정으로 일회용 코드 로그인 한 번 하고 끝이에요.
