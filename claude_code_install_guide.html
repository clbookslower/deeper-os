DEEPER OS · 설치 ①

클로드 코드 설치하기

디퍼 OS(사막여우)는 ‘클로드 코드’라는 자리에서 켜져요. 앱을 깔고, 내 볼트에 연결하는 것까지 — 천천히 같이 해요. 🦊

시작 전 두 가지

① 클로드 유료(Pro 이상) 계정 (무료론 Code 탭이 안 열려요 · 없으면 claude.ai에서 플랜)
② 옵시디언 볼트를 미리 만들어 두기 (아직이면 볼트 만들기 가이드 먼저)

1
앱 다운로드 · 설치

claude.ai/download 에 들어가, 내 컴퓨터에 맞는 걸 받아요. 받은 파일은 보통 ‘다운로드’ 폴더에 있어요.

  • 🪟 윈도우 설치 파일(.exe)을 받아요 → 더블클릭 → 설치 화면이 뜨면 ‘다음/Next’ → ‘설치/Install’ → 끝나면 ‘마침/Finish’.
  • 🍎 맥 .dmg 파일을 받아요 → 더블클릭해 열면 창이 뜨고 → 클로드 아이콘을 ‘Applications(응용 프로그램)’ 폴더로 드래그해 넣어요.

⚠️ “확인되지 않은 앱” 경고가 떠도 괜찮아요 — 공식 앱이에요. 🪟 ‘추가 정보 → 실행’ · 🍎 ‘열기’.

2
앱 실행 · 로그인

설치한 클로드 앱을 열고 → 로그인(Log in) → 브라우저가 열리면 유료(Pro 이상) 계정으로 로그인 → ‘허용/Authorize’ → 앱으로 돌아와요.

3
위쪽 ‘Code’ 탭 + 환경 ‘Local’

화면 위쪽 탭에서 Code 를 눌러요. (코딩하는 곳이 아니라, 내 노트를 다루는 AI를 켜는 자리예요.)

그다음 환경(위치)을 Local 로 골라요 — 이러면 내 컴퓨터의 진짜 폴더에서 일해요.

💡 품질 팁 — 모델은 Opus로

사막여우 답·‘의미 선물 문장’의 깊이는 모델 성능이 좌우해요. 모델 설정에서 Opus 4.8 이상으로. (Opus가 없으면 있는 것 중 제일 좋은 모델로.)

4
내 볼트 폴더 연결하기 ⭐

사막여우가 내 노트를 보려면, “이 폴더에서 일해”라고 가리켜줘야 해요. 이게 옵시디언 ↔ 클로드 코드 연결이에요.

  • Code 탭에서 Select folder(폴더 선택)를 눌러요 → 폴더 고르는 창이 떠요.
  • 옵시디언으로 만든 ‘바로 그 볼트 폴더’를 찾아 선택 → ‘폴더 선택/열기’.
볼트 폴더가 어디 있는지 모르겠다면

옵시디언을 열고 → 좌하단의 볼트 이름(또는 좌측 끝 금고 아이콘)을 눌러 → 폴더 위치를 확인하거나, 파일 위에서 마우스 오른쪽 → ‘시스템 탐색기에서 보기’ 하면 실제 폴더가 열려요. 그 위치를 고르면 돼요.

잘 연결됐는지 확인 — 입력창에 이렇게 쳐보세요:

여기 어떤 폴더야? 안에 뭐가 있어?

→ 내 노트(또는 빈 폴더)를 읽어주면 연결 성공! 🎉

✅ 여기까지 됐으면

이제 클로드 코드가 내 볼트를 보고 있어요. 디퍼 OS를 깔 준비가 다 됐어요.
