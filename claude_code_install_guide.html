DEEPER OS · 설치 ①

클로드 코드 설치하기

디퍼 OS(사막여우)는 ‘클로드 코드’라는 자리에서 켜져요. 앱을 깔고 Code 탭을 여는 것까지 — 5분이면 돼요. 🦊

먼저 준비물 하나

클로드 유료(Pro 이상) 계정이 필요해요. 무료 계정으론 Code 탭이 안 열려요. (없으면 claude.ai에서 플랜 먼저.)

1
앱 다운로드 · 설치

claude.ai/download 접속 → 🪟 윈도우 / 🍎 맥 내 컴퓨터용 파일 받기 → 받은 파일을 실행해서 설치(나오는 대로 ‘계속/설치’).

2
앱 실행 · 로그인

설치한 클로드 앱을 열고, 로그인 → 위에서 만든 유료(Pro 이상) 계정으로 들어가요.

3
위쪽 ‘Code’ 탭 누르기

화면 위쪽 탭에서 Code 를 눌러요. (코딩하는 곳이 아니라, 내 노트를 다루는 AI를 켜는 자리예요.)

4
환경을 ‘Local’ 로

Code 탭에서 환경(위치)을 Local 로 골라요. 이러면 앞으로 내 컴퓨터의 진짜 폴더에서 일해요.

💡 품질 팁 — 모델은 Opus로

사막여우 답·‘의미 선물 문장’의 깊이는 모델 성능이 좌우해요. Code 탭 모델 설정에서 Opus 4.8 이상으로 두세요. (요금제에 Opus가 없으면, 있는 것 중 제일 좋은 모델로.)

✅ 여기까지 됐으면

Code 화면에 Select folder(폴더 선택) 같은 버튼이 보이면 — 클로드 코드 설치 끝! 🎉
